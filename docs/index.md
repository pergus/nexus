# Nexus Documentation

Welcome to the Nexus Django project documentation.

## Overview

Nexus is a Django-based web application with a rich set of UI components and templates. This documentation covers various aspects of the project to help developers understand and work with the codebase effectively.

## Documentation Sections

- **Templates** - Documentation for reusable UI components and template partials

- **Changelog** - Release history and changes

## Getting Started

To view this documentation locally, you can serve it using MkDocs:

```bash
mkdocs serve
```

The documentation will be available at `http://localhost:8000`.
